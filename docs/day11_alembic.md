# Day 11 - Alembic Migrations 🚀

## 📌 What I Learned

### 1. Alembic
- Tool for database migrations
- Tracks DB changes

### 2. Why Needed
- Avoid manual table creation
- Version control for DB

### 3. Commands
- alembic revision --autogenerate
- alembic upgrade head
- alembic downgrade
