# Day 5 - Path vs Query Parameters 🚀

## 📌 What I Learned

### Path Parameters
- Used in URL path
- Required
- Example: /items/1

### Query Parameters
- Used after ?
- Optional
- Example: /products/?name=phone

### Validation
- Query() and Path() used for validation
- Example: price > 0

## Conclusion
Path = resource identifier  
Query = filtering/searching
